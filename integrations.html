E-Commerce Plugins

Add mobile money to your online store

Official NexaPay plugins for the most popular e-commerce platforms. Install from your platform's marketplace or directly from the NexaPay dashboard.

Platform logos: To display partner logos, place the following image files in the same directory as your HTML files: partner-woocommerce.png, partner-shopify.png, partner-prestashop.png, partner-magento.png, partner-opencart.png, partner-wix.png, partner-bigcommerce.png, partner-squarespace.png. Recommended size: 64×64px transparent PNG. Without these files the placeholder emoji icons will be shown.

Mobile & Web SDKs

SDKs for every platform

Official SDKs for iOS, Android, React Native, and Web. All SDKs handle payment initiation, status polling, and webhook signature verification.

📦
Node.js / TypeScript
Full-featured server-side SDK. Supports all endpoints, automatic retries, and webhook signature verification.
npm install nexapay-node
Docs GitHub
🐍
Python
Python 3.8+ SDK with async support (asyncio). Integrates with Django, Flask, and FastAPI.
pip install nexapay-python
Docs
☕
PHP
PHP 7.4+ compatible. Works with Laravel, Symfony, WordPress, and plain PHP. Composer installable.
composer require nexapay/nexapay-php
Docs
📱
React Native
Cross-platform mobile SDK. Supports iOS and Android. Handles deep links, push notifications, and USSD fallback.
npm install nexapay-react-native
Docs
🤖
Android (Kotlin)
Native Android SDK for Kotlin and Java. Includes NexaPay Checkout UI component and low-level payment API wrappers.
implementation 'com.nexapay:sdk:1.x'
Docs
🍎
iOS (Swift)
Native Swift SDK for iOS 14+. Includes SwiftUI components for payment forms and status tracking. SPM compatible.
swift package add nexapay-ios
Docs
Prefer direct API access?
NexaPay exposes a clean REST API with OpenAPI 3.0 documentation. Use any HTTP client in any language — no SDK required. Authentication is a single secret key in the Authorization header.
View API Docs
